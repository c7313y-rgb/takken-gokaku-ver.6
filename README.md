# 宅建14日間 合格トレーナー｜スマホアプリ版

GitHub Pagesへ公開すると、iPhone・Androidのホーム画面へ追加できるPWAとして動作します。

## 公開方法
1. このフォルダ内のファイルをGitHubリポジトリ直下へアップロードします。
2. GitHubの `Settings → Pages` を開きます。
3. `Deploy from a branch`、`main`、`/(root)` を選び保存します。
4. 発行されたHTTPS URLをスマートフォンで開きます。

## スマートフォンへの追加
- iPhone：Safariの共有ボタン →「ホーム画面に追加」
- Android：Chromeの「アプリをインストール」またはアプリ内の「スマホに追加」

## スマホ対応内容
- 片手操作向け固定ボトムナビ
- iPhoneノッチ／ホームインジケータのSafe Area対応
- 44px以上のタップ領域
- 50問模試のスマホ用問題ナビ・固定操作バー
- オフラインキャッシュ
- 学習記録の端末内自動保存
- PWAアイコン、ショートカット、全画面起動

学習記録はブラウザのLocalStorageに保存されます。ブラウザデータの削除や端末変更時には引き継がれません。
